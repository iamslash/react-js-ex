# Build & Run

```bash
# Create a project
$ yarn create vite react-js-vite --template react-ts
$ cd ract-js-vite

# Install node_modules
$ yarn

# Run 
$ yarn dev

# Install ESLint
$ yarn add -D eslint @typescript-eslint/parser
# Run ESLint
$ yarn eslint src/**/*.ts*
# Fix with ESLint
$ yarn eslint src/**/*.ts* --fix
```
